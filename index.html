Software Developer (Innova) - Internship / Summer Job

May 2016 - Sep 2016

Temporary position within the Innova department at Marel. The job was to keep developing a software solution, which had begun as a final project at Reykjavík University, in cooperation with Innova. The project was both research- and development-based. It consisted of exploring how key information, from food processing systems around the world, could be transferred safely, to a single centralized system. The software solution used Microsoft's Azure cloud platform for data streaming, processing and storing. The user interface was developed using the AngularJS and .NET frameworks. The front-end was written in TypeScript and HTML, while the back-end development relied on C#.
